## Tone of Writing
- Professional and academic, appropriate for a university course
- Clear and direct — no filler sentences
- Use prose paragraphs for explanations, bullet lists only for
  structured items
- Each section should be substantive: 2–4 paragraphs minimum
- Never use placeholder text like "Lorem ipsum" or "TBD"

## Custom Colour Palette
- Primary: #1a5276 (deep ocean blue — for headings)
- Accent: #2e86c1 (mid blue — for H2)
- Highlight: #d4ac0d (gold — representing Oceanus Folk genre)
- Background: #f8f9fa (light grey-white)
- Text: #2c3e50 (dark navy)
- Missing info highlight: #e74c3c (red — for blank student IDs and names)

## Missing Info Convention
Any name or ID not yet provided must be wrapped in this exact HTML:
<span class="missing-info">[FULL NAME]</span>
<span class="missing-info">[STUDENT ID]</span>
These render as clearly visible red highlighted boxes on the live site.

## Image Conventions
- All images stored in images/task1/, images/task2/, images/task3/
- Naming: taskN_descriptive_name.png (no spaces)
- Reference in .qmd: ![Caption](images/taskN/filename.png)
