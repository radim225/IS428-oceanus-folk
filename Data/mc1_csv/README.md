## Data Cleaning Notes

- `notable` and `single`: boolean (`True`/`False`), NaN filled as `False`
- `release_date`, `notoriety_date`, `written_date`: string year (e.g. `"2028"`), empty string where missing
- `genre`: `"Unknown"` where missing
- `stage_name`: empty string where missing
- All node IDs are integers matching the `id` column in `lookup_nodes_clean.csv`
